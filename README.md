F-ing-WhiteBoard
================

It's an F-ing Whiteboard app.


Installing:
-----------

1. ./rebar get-deps
2. ./rebar compile generate


Using
-----

### To Run 
  ./rel/fingwb/bin/fingwb start

### To Attach to Running Process
  ./rel/fingwb/bin/fingwb attach

### To Stop 
  ./rel/fingwb/bin/fingwb stop
